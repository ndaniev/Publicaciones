Scenario1: crear
Steps
1. el Lider presiona la opción de proyectos de menú principal
2. SYSTEM muestra la tabla del modulo de proyectos
3. el Lider presiona el botón crear
4. SYSTEM muestra el formulario de crear proyecto
5. el Lider completa los datos que se solicitan para crear un proyecto y presiona la opción "crear y guardar"
6. SYSTEM crea el proyecto

Extension
Steps
5.a.
    1. el Lider presiona en la opción cancelar
    2. SYSTEM cierra la opción de crear y envía al usuario al menú de proyecto

Scenario2:modificar
Steps
1. el Lider presiona la opción de proyectos de menú principal
2. SYSTEM muestra la tabla del modulo de proyectos
3. el Lider elige un proyecto presiona el botón modificar
4. SYSTEM muestra el formulario de modificar proyecto
5. el Lider completa los datos que se solicitan para modificar un proyecto y presiona la opción "modificar y guardar"
6. SYSTEM modifica el proyecto

Extension
Steps
4.a.
    1. el Lider presiona en la opción cancelar
    2. SYSTEM cierra la opción de crear y envía al usuario al menú de proyecto

Scenario3:consultar
Steps
1. elLider presiona en la opción de proyecto de menú principal
2. SYSTEM muestra la tabla del modulo proyecto
3. el Lider busca el proyecto y presiona botón consultar
4. SYSTEM muestra la información del proyecto

Scenario4: eliminar
Steps
1. elLider presiona en la opción de proyecto de menú principal
2. SYSTEM muestra la tabla del modulo proyecto
3. el Lider se ubica sobre el proyecto y presiona en la opción eliminar
4. SYSTEM muestra el aviso si desea eliminar la competencia
5. if el Lider presiona la opción de aceptar
    5.1. SYSTEM verificara permiso pidiendo la contraseña del Lider
    5.2. while la contraseña sea verdadera
        5.2.1. SYSTEM borrar el proyecto
         end while
   end if

Extension
Steps
4.a. el Lider presiona en el botón cancelar
    1. SYSTEM cierra las opciones de eliminar y envía al usuario al menú de proyecto
5.a. el Lider presiona en opción de cancelar
    1. SYSTEM cierra la opción de verificación
