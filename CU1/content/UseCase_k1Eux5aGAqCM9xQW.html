Scenario1: consultar limitaciones
Steps
1. el Lider presiona la opción ambientes del menú principal
2. SYSTEM muestra la ventana de ambientes
3. el Lider presiona la opción limitaciones del ambiente
4. SYSTEM muestra el formulario de las limitaciones con su respectivo ambiente
5. el Lider elige un ambiente presiona consultar limitación
6. SYSTEM muestra las limitaciones del ambiente elegido.

Scenario2: crear limitacion
Steps
1. el Lider presiona la opción de ambientes de menú principal
2. SYSTEM muestra la tabla del modulo de ambientes
3. el Lider presiona la opción limitar ambiente
4. SYSTEM muestra el formulario de los ambientes
5. el Lider elige un ambiente presiona el botón crear limitación
6. SYSTEM muestra el formulario de crear limitaciones del ambiente
7. el Lider completa los datos que se solicitan para crear una limitación y presiona la opción "crear y guardar"
8. SYSTEM crea limitación a un ambiente

Extension
Steps
7.a.
    1. el Lider presiona en la opción cancelar
    2. SYSTEM cierra la opción de crear y envía al usuario al menú principal

Scenario3: modificar limitaciones
Steps
1. el Lider presiona la opción de ambientes de menú principal
2. SYSTEM muestra la tabla del modulo de ambientes
3. el Lider elige un ambiente presiona el botón modificar limitación
4. SYSTEM muestra el formulario de modificar limitación
5. el Lider completa los datos que se solicitan para modificar una limitación y presiona la opción "modificar y guardar"
6. SYSTEM modifica la limitación

Extension
Steps
4.a.
    1. el Lider presiona en la opción cancelar
    2. SYSTEM cierra la opción de crear y envía al usuario al menú de ambientes

Scenario4: eliminar limitaciones
Steps
1. elLider presiona en la opción de ambientes de menú principal
2. SYSTEM muestra la tabla del modulo ambientes
3. el Lider elige un ambiente presiona consultar limitación
4. SYSTEM muestra las limitaciones del ambiente elegido.
5. el Lider se ubica sobre la limitación del ambiente y presiona en la opción eliminar
6. SYSTEM muestra el aviso si desea eliminar la limitación del ambiente
7. if el Lider presiona la opción de aceptar
    7.1. SYSTEM verificara permiso pidiendo la contraseña del Lider
    7.2. while la contraseña sea verdadera
        7.2.1. SYSTEM borrar el la limitación
         end while
   end if

Extension
Steps
6.a. el Lider presiona en el botón cancelar
    1. SYSTEM cierra las opciones de eliminar y envía al usuario al menú de ambientes
7.a. el Lider presiona en opción de cancelar
    1. SYSTEM cierra la opción de verificación
